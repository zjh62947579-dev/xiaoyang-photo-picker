# ⚠️ Windows + NVIDIA 显卡手动安装建议：
#
# 这个项目现在默认按「Windows 独显优先」来配依赖。
# `requirements.txt` 里把 ONNX Runtime 切到了 GPU 版；但 `torch/torchvision`
# 不能只靠这一份文件从 PyPI 装，否则很容易又装回 CPU 版。
#
# 推荐顺序：
#   1) python -m pip install -r requirements.txt
#   2) python -m pip install torch torchvision --index-url https://download.pytorch.org/whl/cu128
#
# 装完后验证：
#   python -c "import torch, onnxruntime as ort; print(torch.__version__, torch.cuda.is_available(), torch.version.cuda); print(ort.get_available_providers())"
#
# 如果你的环境拉不下 cu128，可把上面的 `cu128` 换成 `cu126` 再试。
#
# ⚠️ OpenCV 注意事项（手动 pip install 走这条路径必读）：
#
# `insightface` 和 `pyiqa` 等会偷偷把 `opencv-python` 拉进来作传递依赖。
# 三个 OpenCV 包（opencv-python / opencv-python-headless / opencv-contrib-python）
# 共存会让 cv2 主包覆盖 contrib 的子模块——cv2.saliency 直接失效，
# 整个 expert 模式跑不起来。
#
# 装完 requirements 后，请务必执行：
#   pip uninstall -y opencv-python opencv-python-headless
#   pip install --force-reinstall --no-deps "opencv-contrib-python>=4.9"
#
# 一键启动器（scripts/launcher.py）会自动做这件事，手动模式需要自己做。
